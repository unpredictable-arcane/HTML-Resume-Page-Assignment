## Project Overview

The resume includes the following sections:

- Header (Name & Title)
- About Me
- Skills
- Experience (Table)
- Projects
- Education (Table)
- Contact Details
- Footer

The entire page is built using only HTML, without CSS or JavaScript, as per assignment guidelines.
